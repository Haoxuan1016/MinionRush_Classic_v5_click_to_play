# Minion Rush V5.7 — Full Offline Repack

A repack of the classic **Minion Rush v5.7** that bundles all game data directly inside the APK, producing a single **~600 MB** installable package with no separate download step.

## Download

The full APK is available on the **[Releases](../../releases)** page. Download the latest release, install it on your device, then turn off internet before launching.

## About

This build is based on the classic offline Minion Rush experience. Original online services are no longer available, so the game is intended to be played **fully offline**.

Instead of downloading game data on first launch, this repack embeds the assets inside the APK. On install, everything needed to run the game is already present.

## Requirements

- **Android device** (ARM 32-bit / `armeabi-v7a`)
- **Turn off internet** (Wi‑Fi and mobile data) before playing — the game is meant to run in offline / airplane mode

## What Changed

This repository does **not** re-create the game or its assets. The only addition here is an **auto-load script** that handles loading the bundled data automatically, so no manual setup or extra download is required.

All original game content comes from the credited sources below.
